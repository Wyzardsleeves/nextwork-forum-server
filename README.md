## NextWork Forum App

### Get started

Run npm install

``` bash
npm install
```

### Technologies used

- [**React**](https://react.dev/) as a client. (I would not integrate it like this in production of course).
- [**Node/Express**](https://expressjs.com/en/) as a server.
  - I understand you guys use GoLang; however, Christopher told me you would also appreciate Node.
- [**Firebase**](https://firebase.google.com/) as a database and for authentication. 
  - I didn't think we needed anything too fancy.

### Process / Notes

- I allowed VS Code's copilot to autocomplete big functions of course I needed to make changes.
- When installing packages I look for the latest trends anyway to make sure there's note a new way to do things that I don't know about. 
- I started by adding requests in the client for what I needed from the backend and treated it like a bucketlist.

### What else I would do

- Set up roles.
- Set up a Captcha so that robots can't post spam
- Allow users to
  - Edit their avatar
